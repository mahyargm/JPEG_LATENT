= List of betas 

The current drectory contains subdirectories with list of `beta`s for each image and rate. 

A name of the subdirectory corresponds to a name of operation point. Path of the subdirectory set in a encoder's configuration file of each operation point.

Each directory has two files:
- `betas_tools_off.txt` is a file with a list of `beta`s for tools-off configuration (see it [here](../tools_off.json))
- `betas_tools_on.json` is a file with a list of `beta`s for tools-on configuration (see it [here](../tools_on.json))
